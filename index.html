Projects

Sentiment Analysis across Social Media Platforms

The Sentiment Analysis Across Social Media Platforms project analyzes public sentiment from social media using natural language processing and machine learning. It classifies sentiments as positive, negative, or neutral and provides a real-time dashboard for trend visualization, offering valuable insights for brand monitoring and marketing effectiveness.

Smart Spoon for people with Parkinson's disease

The Smart Spoon for Parkinson's disease uses an Arduino Nano to help individuals keep their spoon level despite hand tremors, preventing spills. Designed and tested for stability, it ensures the spoon remains upright, enhancing the dining experience and improving quality of life for those with Parkinson's disease.

RFID- based attendance system

The RFID-based attendance system, built with PHP, CSS, and JavaScript, uses WIFI for tracking. When an RFID card is swiped, the system logs the arrival time and, upon a second swipe, records the departure time. It utilizes the NodeMCU ESP8266 and MF-RC522 module to transfer data to a PHP web app and store it in a database.
